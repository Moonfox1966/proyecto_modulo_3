# Documentación Técnica
## Sistema de Registro de Pasajeros en un Hotel

Este sistema permite gestionar el registro de pasajeros de un hotel mediante
un menú por consola desarrollado en Python. Administra datos de pasajeros como
nombre, documento de identidad, nacionalidad, número de habitación, fechas de
ingreso y salida y estado de la estadía.

El sistema permite crear, listar, buscar, actualizar y eliminar registros,
utilizando estructuras de datos como listas, diccionarios, conjuntos y tuplas.
El código está organizado en módulos para facilitar su mantención y lectura.



	•	main.py: menú principal y ejecución del sistema.
	•	modulos/: lógica del sistema separada por responsabilidad.
	•	entradas.py: input del usuario.
	•	validaciones.py: validaciones de datos.
	•	operaciones.py: funciones principales.
	•	reportes.py: salidas, resúmenes o cálculos.
	•	data/: archivos .csv o .txt de prueba.
	•	docs/: documentación solicitada.
	•	presentacion/: presentación final.
